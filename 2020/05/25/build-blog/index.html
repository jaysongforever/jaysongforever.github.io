如何免费搭建个人主页

2020-05-25
建站
  1. 1.下载git,正常安装软件获取秘钥
  2. 2.注册Github并配置SSH Keys，测试连接
  3. 3.创建仓库，push代码
  4. 4.打开Github pages功能

准备工具:

1.git官网：https://git-scm.com

2.github官网：https://github.com

3.github pages主页：https://pages.github.com

1.下载git,正常安装软件获取秘钥

  • 下载并安装Git：正常安装Git后进入桌面,右键鼠标，点击进入Git Bash Here

  • 检查本机是否有ssh key：输入命令cd ~/.ssh，如果没有则提示： No such file or directory，如果有则进入~/.ssh路径下，输入ls查看当前路径文件

  • 生成新的SSH Keys（已配置过SSH Keys跳过此步）：

    ssh-keygen -t rsa -C "xxxxx@xxxxx.com"

    按照提示完成三次回车，即可生成 ssh key。通过查看id_rsa.pub 文件内容，获取到你的 public key，输入命令

    cat ~/.ssh/id_rsa.pub右键复制public key保存，后面要用

2.注册Github并配置SSH Keys，测试连接

  • 配置SSH Keys：成功注册后点击头像，进入Settinngs，点击SSH and GPG keys，title随便填，Key里面填之前获取到的public key，然后Add

  • 配置账户，设置username和email：github每次commit都会记录他们，

    git config --global user.name "name"

    “”里面替换为你的GitHub用户名

    git config --global user.email "123@126.com"

    “”里面替换为你的GitHub注册邮箱

  • 测试连接：输入命令ssh -T git@github.com,出现如下提示说明与Github连接成功

3.创建仓库，push代码

  • 创建远程仓库：点击github页面右上角头像旁边的+号，选择New repository，新建仓库

  • clone远程仓库到本地：复制ssh url，进入本地你要放置此项目的文件夹下，并且在此文件夹下打开命令行工具（点击左上角【文件】，再点击打开Windows PowerShell），使用命令git clone + 刚才复制的url，如下

    git clone git@github.com:jaysongforever/testweb.git

    稍等片刻，即可将远程仓库clone到本地，本地新产生的testweb文件夹，可看成是你的项目文件夹，需在此文件夹内进行开发，并且index.html文件需放置在项目文件夹的根目录下

  • push本地代码到远程仓库：

    在项目文件夹内打开命令行工具，输入命令

    git add . //推到缓冲区

    git commit -m "此次操作备注" //从缓冲区推到本地仓库

    git push //从本地仓库推到远程仓库

    首次push需要验证github账户，输入账户密码登录即可

4.打开Github pages功能

  • push成功后，本地代码推到了线上，刷新github项目所在仓库，即可看见所推代码

  • 进入此仓库的Settings，找到Github Pages，勾选master branch，刷新后出现如下url，即为你项目的线上访问地址https://jaysongforever.github.io/testweb/，后续更新网页内容，直接push代码网页可自动更新
